Adobe® Acrobat® Reader für Windows - Liesmich


Willkommen zur Liesmich-Datei für Adobe® Acrobat® Reader 5.0.5. Adobe Acrobat Reader ist das kostenlose Anzeigeprogramm, das als Begleitprogramm zu Adobe Acrobat 5.0 erscheint. Mit Acrobat Reader können Sie PDF-Dateien (Portable Document Format) anzeigen, in ihnen navigieren und sie drucken. Adobe Acrobat 5.0 ist ein Komplettpaket zum Erstellen, Erweitern, Überprüfen, Bearbeiten und Freigeben von Informationen im PDF-Format. Wissenswertes über Adobe Acrobat 5.0 erfahren Sie auf unserer Website unter
http://www.adobe.de/products/acrobat

Diese Liesmich-Datei enthält Informationen zu folgenden Themen:

Systemanforderungen

Installationsanweisungen

Systemanforderungen

Systemanforderungen für Acrobat Reader 5.0.5 für Windows

- Prozessor der Pentium®-Klasse
- Microsoft® Windows® 95 OSR 2.0, Windows 98 Second Edition, Windows Millennium Edition, Windows NT 4.0 mit Service Pack 5 oder 6 (Service Pack 6 empfohlen), Windows 2000 oder Windows XP
- 16 MB RAM (32 empfohlen)
- 20 MB verfügbarer Festplattenspeicher
- Zusätzlich 30 MB Festplattenspeicher für asiatische Schriften (optional)

Unterstützung für Web-Browser

PDF-Dateien können in den folgenden Web-Browsern mit Acrobat oder Acrobat Reader angezeigt werden:
Internet Explorer 4.x
Internet Explorer 5.0 oder 5.5
Netscape Navigator 4.x, 6.1
America On Line 5.0 oder 6.0

Installationsanweisungen

Installieren und Deinstallieren von Acrobat Reader 5.0.5 für Windows

Führen Sie zum Installieren von Acrobat Reader 5.0.5 folgende Schritte aus:

- Stellen Sie sicher, daß Sie mindestens 20 MB freien Speicherplatz zur Verfügung haben.
- Deaktivieren Sie vor dem Beginn der Installation jegliche Anti-Virus-Software.
- Installieren Sie Reader 5.0.5.

Führen Sie zum Deinstallieren von Acrobat Reader 5.0.5 folgende Schritte aus:
- Wählen Sie aus dem Start-Menü die Optionen "Einstellungen", "Systemsteuerung", "Software"
- Wählen Sie Adobe Acrobat Reader 5.0 aus, und klicken Sie auf die Schaltfläche "Hinzufügen/Entfernen..."

Hinweis: Bei der Deinstallation von Acrobat Reader 5.0.5 werden Acrobat 5.x, Acrobat 4.x und Acrobat 3.x ebenfalls entfernt, wenn diese Programme auf Ihrem System installiert sind, da diese Programme gemeinsame Dateien verwenden.

Installationsprobleme

Deaktivieren Sie vor dem Beginn der Installation jegliche Anti-Virus-Software. Einige Gerätetreiber können ebenfalls einen Konflikt mit dem Installationsprogramm von Acrobat Reader verursachen. Wenn Sie Windows 95 oder Windows 98 im abgesicherten Modus oder Windows NT im VGA-Modus starten, lädt Windows nur die unbedingt erforderlichen Standardgerätetreiber.

Treten beim Ausführen des Installations- bzw. Deinstallationsprogramms von Acrobat Reader 5.0.5 Probleme auf, löschen Sie die Datei "Uninst.isu" aus dem Ordner "Programme\Gemeinsame Dateien\Adobe\Acrobat 5.0\98" bzw. "...\NT", und installieren bzw. deinstallieren Sie Acrobat Reader 5.0.5 erneut.

Adobe SVG Viewer 2.0

Bei der Installation von Adobe Acrobat Reader 5.x wird automatisch Adobe SVG Viewer 2.0 installiert. Um weitere Informationen über Adobe SVG Viewer 2.0 anzuzeigen, öffnen Sie mit Ihrem Browser die Liesmich-Datei "ReadMe.html". Diese Datei finden Sie im Plugin-Ordner Ihres Browsers.

Installieren der asiatischen Sprachdateien für Acrobat Reader 5.x

Für Acrobat Reader 5.x sind asiatische Sprachdateien verfügbar. Sie können mit jeder beliebigen Sprachversion von Acrobat Reader 5.x Adobe PDF-Dateien mit chinesischem (traditionell und vereinfacht), japanischem oder koreanischem (CJK) Text anzeigen. Sie brauchen einfach nur das entsprechende asiatische Schriftenpaket zur Verwendung mit Acrobat Reader zu installieren. Die Installationsprogramme zur Unterstützung für asiatische Sprachen finden Sie auf der Produkt-CD von Adobe Acrobat 5.x oder auf der Website von Adobe unter:
http://www.adobe.com/prodindex/acrobat/cjkfontpack.html

Adobe Acrobat Reader 5.0.5 - Zusätzliche Informationen

Adobe Acrobat 5.0.5 bietet einen verbesserten Zugriff, Online-Kommentare, digitale Unterschriften, PDF-Erstellung, Formularerstellung, Formulardateneingabe, Installation, Überdruckvorschau, Rechtschreibprüfung sowie TouchUp-Funktionen. In Adobe Acrobat 5.0.5 sind ebenfalls Probleme behoben, die im Zusammenhang mit der Anzeige und dem Drucken bestimmter PDF-Dateien festgestellt wurden. Benutzern von Adobe Acrobat 5.0.5 für Windows steht außerdem eine erweiterte Unterstützung für Windows XP und Netscape 6.0 zur Verfügung. Die Produkt-CD von Acrobat 5.0.5 beinhaltet zudem die Zusatzmodule Make Accessible und Paper Capture. Verwenden Sie das Zusatzmodul Make Accessible, damit auf eine PDF-Datei behindertengerecht zugegriffen werden kann. Mit Hilfe des Zusatzmoduls Paper Capture können Sie elektronische Dokumente in PDF-Dateien konvertieren (einschließlich OCR - optischer Zeichenerkennung).

Die Suche nach bestimmten Zeichen wird nicht unterstützt
Adobe Acrobat Catalog unterstützt nicht die Indizierung der WinAnsi-Zeichen 0127 bis 0158. Zu diesen Zeichen gehören u. a. auch Währungssymbole (beispielsweise das Dollar-, Euro- und Yen-Zeichen). Adobe Acrobat Catalog unterstützt zudem nicht die Indizierung von Dokumenten, die mit aktivierten Ligaturen, kleinen römischen Ziffern oder Kapitälchen formatiert sind. Da diese Zeichen nicht indiziert werden können, ist es nicht möglich, die Volltextsuche zu verwenden, um nach diesen Zeichen in einer PDF-Datei zu suchen.[443269,418016]

Drucken von sortieren Exemplaren ist unter Windows 2000 nicht möglich.[415358, 418016]

Beim Verwenden des über die rechte Maustaste verfügbaren Druckbefehls wird die PostScript-Level-Einstellung des vorherigen Druckauftrags unter Windows NT und Windows 2000 verwendet
Wenn Sie ein Dokument in Acrobat verwenden und es auf einen Drucker mit PostScript-Level 3 drucken, wird der von Acrobat verwendete Level standardmäßig auf PostScript Level 3 eingestellt. Wenn Sie Acrobat beenden und als Standarddrucker einen PostScript Level 2-Drucker festlegen und dann auf eine PDF-Datei mit der rechten Maustaste klicken, um einen Druck anzufordern, wird der Druckauftrag abgebrochen, da PostScript Level 3 und nicht Level 2 eingerichtet ist. Um dieses Problem nach Änderung des Standarddruckers zu vermeiden, öffnen und drucken Sie in Acrobat ein Dokument, bevor Sie den über die rechte Maustaste verfügbaren Befehl zum Drucken verwenden.[416598, 418016]

Problembehebung


Verwenden von Acrobat 4.x oder Acrobat Reader 4.x zum Anzeigen und Drucken von in Acrobat 5.0 erstellten Dateien
 Sie können in Acrobat 5.x erstellte PDF-Dateien mit Acrobat 4.x oder Acrobat Reader 4.x anzeigen und drucken; beachten Sie jedoch folgendes:
-PDF-Dateien mit einer 128-Bit-Verschlüsselung können in Acrobat 4.x nicht geöffnet werden,
-Transparenz wird in Acrobat 4.0 nicht unterstützt; PDF-Dateien mit entsprechenden Grafiken können nur in Acrobat 5.0 einwandfrei angezeigt werden,
-Gescannte PDF-Dateien, die JBIG2-Komprimierung verwenden, können in Acrobat 4.x nicht geöffnet werden,
-Mit Acrobat 5 kann jede Schriftart Ihres Systems in einer mit dem Freier Text-Werkzeug erstellten Anmerkung verwendet werden. In Acrobat 4.x waren hierfür nur die PostScript Roman-Schriften und die Heisei Japanese-Schriften zulässig. Wenn eine Anmerkung mit freiem Text mit japanischen Schriftzeichen in Acrobat 5.x erstellt wird und dabei nicht die Japanese Heisei-Schriften verwendet werden, wird diese Textanmerkung in Acrobat 4.x nicht einwandfrei angezeigt, [397619]
-PDF-Dateien, die neue JavaScript-Operatoren enthalten, können Fehlermeldungen an die JavaScript-Konsole ausgeben, wenn sie in Acrobat 4.0 geöffnet werden,
-PDF-Dateien, deren Schriften neue vordefinierte CMaps verwenden, können unter Umständen nicht einwandfrei in Acrobat 4.0 geöffnet werden,
-Einige in Acrobat 5.x erstellte Formulare erzeugen eventuell FDF-Daten, die von den CGI-Skripts des Servers, die mit dem Acrobat 4.0 FDF-Toolkit erstellt wurden, nicht korrekt analysiert werden können,
-Acrobat-Formularfelder in Acrobat 4.x unterstützten nur eine begrenzte Anzahl von Schriften. Während Sie mit dem Formular-Werkzeug in Acrobat 5.x jede beliebige Schrift festlegen können, die das Bearbeiten und Einbetten in einem Formularfeld ermöglicht, können Benutzer in Acrobat 4.x solche Formulare nicht ausfüllen. Berücksichtigen Sie dies beim Erstellen von Formularen, die möglicherweise mit Hilfe von Acrobat 4.x angezeigt werden. [397754, 312623,410665]

Verwenden der Werkzeugleiste
Die Werkzeugleisten von Acrobat Reader können verschoben oder am oberen Bildrand verankert werden, außerdem kann das Erscheinungsbild verändert werden. Verschobene Werkzeugleisten können nicht kombiniert werden, indem Sie die Titelleiste der Werkzeugleiste ziehen; verwenden Sie statt dessen die Ziehgriffe der Werkzeugleiste.[398112]

Anzeigen von PDF-Dateien mit Internet Explorer und Netscape
Wenn Sie versuchen, relativ große PDF-Dateien anzuzeigen, erhalten Sie möglicherweise eine Fehlermeldung. Es kann auch vorkommen, daß Acrobat Reader leere Seiten anzeigt oder nicht mehr reagiert. Fragen Sie Ihren System- oder Netzwerkverwalter, um sicherzugehen, daß Sie die Apache-Version 1.3.17 oder neuer verwenden. Mit einer früheren Version von Apache weist Acrobat Reader eventuell die oben genannten Symptome auf.[417553]

Zugriff
Eine erweiterte Version von Acrobat Reader für Windows bietet Unterstützung für Sehbehinderte. Um festzustellen, ob Ihre Kopie von Acrobat Reader über diese Zugriffsfunktionen verfügt, suchen Sie nach dem Zusatzmodul MSAA im Dialogfeld unter "Hilfe > Info über Adobe Acrobat-Zusatzmodule".

Zugriff - Verwenden des Befehls "Umfließen" bei Bildschirmleseprogrammen
Um zuverlässige Ergebnisse bei Bildschirmleseprogrammen zu erhalten, sollten nur die Dokumente umflossen werden, die nicht einzeln an das Bildschirmleseprogramm weitergegeben werden. Sie können die Dokumentzustellung an Sprachausgabeprogrammen steuern, indem Sie die Inhaltsübermittlung im Fenster für die Zugriffsgrundeinstellungen festlegen. [418683]

America On Line - Anzeigen von PDF-Dateien unter Windows 95 mit America On Line 6
Wenn Sie unter Windows 95 in America On Line 6 mit installiertem Internet Explorer 5.5 auf eine Verknüpfung zu einer PDF-Datei klicken, kann der Ladevorgang zwei bis vier Minuten dauern. Verwenden Sie Internet Explorer oder Netscape, wenn Sie diese Möglichkeit haben.[412577]

America On Line - Wenn Sie den Befehl "Datei > In America On Line 6.0 öffnen" verwenden, wird die PDF-Datei in einem Texteditor geöffnet.
Wenn Sie die einzelnen Schritte durchlaufen, wird ein Dialogfeld angezeigt, das Ihnen mitteilt, daß die PDF-Datei kein anerkannter Dateityp ist und in einem Texteditor geöffnet wird. Zum Anzeigen einer PDF-Datei, die sich auf Ihrer Festplatte oder Ihrem LAN befindet, wählen Sie sie einfach im Windows Explorer aus. Acrobat Reader wird gestartet und lädt die Datei.[405406, 406627]

Chinesische, koreanische und japanische (CJK) Acrobat-Ersatzschriften
Sie sollten nicht versuchen, Adobe Type Manager zu aktivieren und die von Acrobat installierten CJK-Ersatzschriften zu verwenden (zum Beispiel "KozMinPro-Regular-Acro.otf"). Dies kann zu unerwarteten Ergebnissen führen.[390851]

Anzeige von CJK-Text in Acrobat Reader-Dialogfeldern
Auf einem lateinischen System kann in den Dialogfeldern von Acrobat Reader kein chinesischer, koreanischer und japanischer (CJK) Text angezeigt werden. Acrobat Reader benötigt ein CJK-System mit der entsprechenden CJK-Systemschrift, um CJK-Zeichen in den Dialogfeldern anzeigen zu können.[395326]

Anzeige von CJK-Text mit aktivierter Überdruckvorschau
Wenn eine PDF-Datei, die besonders komplizierte CJK-Zeichen enthält, mit aktivierter Überdruckvorschau angezeigt wird, erscheint der Text unter Umständen unschärfer als mit deaktivierter Überdruckvorschau.[416395]

CJK-Anzeige von fettformatierten Text bei Vergrößerung
Einige fettformatierte CJK-Zeichen werden möglicherweise verschwommen angezeigt, wenn die Ausgangsanwendung den fettformatierten Text "synthetisch" erstellt hat. Wenn in den Grundeinstellungen für die Anzeige die Optionen "Text glätten" und "CoolType verwenden" deaktiviert wurden, werden bei der vergrößerter Dokumentanzeige u. U. weiße Bereiche oder Risse angezeigt. Acrobat 4.x schreibt ein Zeichen viermal beim Format "Fett", und bei einigen Dokumenten kann derartiger Text, statt fettformatiert, als vier Textzeilen angezeigt werden. In Acrobat 5.x wird hierzu ein "synthetischer" Effekt verwendet. Adobe empfiehlt, in den Grundeinstellungen für die Anzeige die Optionen "Text glätten" und "CoolType verwenden" zu aktivieren, um weiße Bereiche oder Risse beim Vergrößern von fettformatiertem Text zu vermeiden. [418447]

CJK-Formulare
 Wenn Sie eine PDF-Datei mit CJK-Formularen öffnen, deren CJK-Zeichen anders kodiert sind als die, mit denen das Formular erstellt wurde, werden die CJK-Formularfelder auf einem System, bei dem die Unterstützung für asiatische Sprachen nicht installiert ist, nicht korrekt angezeigt. Die meisten Formularfelder werden gar nicht angezeigt. Um dies zu vermeiden, installieren Sie die Unterstützung für asiatische Sprachen auf den Systemen, bei denen CJK-Formulare angezeigt und verwendet werden. Weitere Informationen dazu finden Sie im Abschnitt "Installieren der asiatischen Sprachdateien für Acrobat Reader 5.x" in dieser Liesmich-Datei. [412829]

Online-Erstellen von Adobe PDF und Adobe PDF Online-Volltextsuche erfordern Winsock2
 Diese Funktionen, die über das Menü "Werkzeuge" zugänglich sind, erfordern Winsock2. Wenn Winsock2 nicht verfügbar ist, erscheinen diese Funktionen nicht. Außer unter Windows 95 ist Winsock2 auf Windows-Betriebssystemen verfügbar, weil es entweder gleichzeitig mit Internet Explorer 4 oder Internet Explorer 5 installiert wird. Um Web-Dienste unter Windows 95 verfügbar zu machen, installieren Sie Winsock2 (erhältlich bei Microsoft).[390722]

Grundeinstellung zum Extrahieren von Bildern
In der Acrobat-Hilfe wird im Abschnitt "Festlegen von Acrobat-Grundeinstellungen" die Grundeinstellung zum Extrahieren von Bildern nicht korrekt definiert. In der Hilfe wird die Grundeinstellung folgendermaßen beschrieben: "Definiert die minimale Größe eines Bildes zur Anzeige auf dem Bildschirm." Die Grundeinstellung "Bilder extrahieren" definiert für Acrobat die minimale, erforderliche Größe eines Bildes, damit es in Acrobat aus einer PDF-Datei entnommen werden kann. [417231]

Dialogfeld "Suchen" - Verwenden des Suchen-Befehls für Unicode-Zeichen
Sie können den Suchen-Befehl verwenden, um innerhalb eines PDF-Dokuments nach Unicode-Zeichen zu suchen. Hierzu fügen Sie vor dem zu suchenden Unicode-Zeichen "\u" ein. Wenn Sie z. B. nach dem dem ASCII-Zeichen "A" entsprechenden Unicode-Zeichen suchen, geben Sie "\u0041" im Dialogfeld "Suchen" ein. Für einige CJK-Zeichen, beispielsweise Unicode "4e01", können Sie "\u4e01" festlegen.[413632]

Acrobat 5-Unterstützung für den zusätzlichen Hong Kong-Zeichensatz (Hong Kong Supplemental Character Set)
Die zuständige Sonderverwaltungsbehörde in Hongkong (Hong Kong Special Administrative Region Government) hat vor kurzem den zusätzlichen Hong Kong-Zeichensatz (HKSCS) vorgestellt. Acrobat 5 ermöglicht Ihnen, PDF-Dateien mit HKSCS-Zeichen auf allen von Acrobat 5.x unterstützten Windows-Betriebssystemen (siehe Abschnitt zu den Systemanforderungen) anzuzeigen und zu drucken. Wenn Sie PDF-Dateien anzeigen möchten, die Text enthalten, der den Hong Kong-Zeichensatz verwendet, installieren Sie die Unterstützung für den Zeichensatz für traditionelles Chinesisch, indem Sie die benutzerdefinierte Installation wählen oder das Sprachpaket für traditionelles Chinesisch installieren. Mit Hilfe von Distiller 5.x können Sie PDF-Dateien mit HKSCS-Zeichen auf den Betriebssystemen Windows ME und Windows 2000 erstellen; andere Windows-Betriebssysteme werden nicht unterstützt. Bevor Sie Distiller 5.x zum Erstellen solcher Dokumente verwenden können, müssen Sie das über die folgende Website der Behörden in Hongkong verfügbare Kit installieren: http://www.info.gov.hk/digital21/eng/hkscs/download.html [418559]

Hong Kong-Zeichensatz ist keine Standardsystemschrift
Acrobat Reader 5.x unterstützt den Hong Kong-Zeichensatz für Text in einer PDF-Datei. Er stellt eine Erweiterung des Zeichensatzes von Adobe für traditionelles Chinesisch dar. Wenn Sie PDF-Dateien anzeigen möchten, die Text enthalten, der den Hong Kong-Zeichensatz verwendet, installieren Sie die Unterstützung für den Zeichensatz für traditionelles Chinesisch, indem Sie die benutzerdefinierte Installation wählen oder das Sprachpaket für traditionelles Chinesisch installieren. Da der Hong Kong-Zeichensatz nicht Teil der Standardsystemschriften ist, können Sie keine Kommentare oder Lesezeichen mit Hong Kong-Zeichen erstellen.[407877]

Installieren von Acrobat 3.x oder 4.x nach der Installation von Acrobat Reader 5.x
Wenn Sie Acrobat 3.x oder 4.x nach der Installation von Acrobat Reader 5.x installieren, wird Acrobat Reader jedesmal automatisch gestartet, wenn im Windows Explorer ein Ordner geöffnet wird. Um dieses Problem zu vermeiden, deinstallieren Sie Acrobat Reader 5.x, bevor Sie eine ältere Version von Acrobat installieren.[400589]

Zusatzmodul Movie
 Wenn Sie ein Movie in einem frei schwebenden Fenster anzeigen, können Sie das Movie jederzeit durch Drücken der Escape-Taste anhalten.[410153]

Movie-Zusatzmodul - Remote-URL erfordert QuickTime 4 oder höher
Wenn Sie das Movie-Werkzeug verwenden möchten, um über eine Verknüpfung zu einem externen URL Filme anzuzeigen, sollten Sie sicherstellen, daß QuickTime 4 oder höher auf Ihrem System installiert ist. Zum Installieren von QuickTime 4 wählen Sie die benutzerdefinierte Installation von Acrobat, ändern dann die Auswahl für die Zusatzelemente so, daß QuickTime 4 enthalten ist. Das Installationsprogramm für QuickTime von Apple wird gestartet. Ihre Produkt-CD umfaßt ebenfalls das Installationsprogramm für QuickTime 4 von Apple, das Sie im Utilities-Ordner finden. [418201]

Netscape 6
Acrobat Reader 5.x unterstützt Netscape 4.x- und Netscape 6.1-Browser und installiert ein Zusatzmodul für diese Browser. Durch dieses Zusatzmodul wird es Benutzern ermöglicht, Adobe PDF-Dateien herunterzuladen, anzuzeigen, darin zu navigieren und sie zu drucken sowie PDF-Formulare im Browser auszufüllen. In Netscape 6.0 müssen PDF-Dateien komplett heruntergeladen werden, um sie anzeigen zu können. Außerdem können PDF-Formulare in Netscape 6.0 nicht ausgefüllt werden, wenn sie im Browser angezeigt werden. Verwenden Sie, falls möglich, Netscape 6.1.

Nvidia TNT graphics card with True (32-bit) color set
If you are using a Nvidia TNT graphics card and your display is set to use True (32-bit) color, you may experience a general protection fault crash in Module NVDISP.drv. The crash will occur if the Opacity setting in the Comments preferences is set to less than 100% and there are one or more Comment popup windows overlapping. You can avoid the problem by setting the Opacity in the Comments Preferences to 100% or by setting the display to 16-bit color (in the Display Control Panel). Please check the Nvidia web site for a driver update that may solve this problem. [419590]

Drucken von ImageType 4-Dateien auf Lexmark-Druckern
ImageType 4 ist eine Funktion von PostScript 3. Während Anwendungssoftware nur selten ImageType 4-Dateien erstellt, enthält eine PDF-Datei, die mit Acrobat WebCapture aus einer GIF-Datei mit transparenten Pixeln erstellt wurde, ImageType 4.
Wenn Dateien, die ImageType 4 enthalten, auf den unten aufgelisteten Lexmark-Druckern gedruckt werden, reagiert der Drucker unter Umständen nicht mehr, und es erfolgt keine Ausgabe. Dies gilt für folgende Druckermodelle von Lexmark:
-Lexmark Optra C710 PS3, PS Version 3010.010, PS revision 2000411
-Lexmark Optra C710 PS3, PS Version 3010.010, PS revision 990917
Um dieses Problem zu vermeiden, haben Sie zwei Möglichkeiten:
1. Wählen Sie im Drucken-Dialogfeld die Option "Als Bild drucken". Das erzielte Ergebnis ähnelt dem, was Sie auf dem Bildschirm sehen.
2. Wenn Sie Windows als Betriebssystem verwenden, können Sie die Einstellung unter "Druckmethode" von "Level 3" zu "Level 2" ändern. Für Macintosh gibt es jedoch kein Eingabefeld für die Druckmethode. Sie können das gleiche Ergebnis erzielen, indem Sie die PPD-Datei so bearbeiten, daß diese Level 2 wiedergibt; ändern Sie z. B.:
*PSVersion: "(3010.010)"
in:
*PSVersion: "(2014.700)"
Ändern Sie die PPD-Datei nicht, wenn mit der Option "Als Bild drucken" das gewünschte Ergebnis erzielt wird. Soll die Datei mit Level 2 drucken, kann ImageType 4 gedruckt werden, jedoch geht die Transparenz verloren.[416539]

Drucken von Schattierungen auf HP4050N-Druckern
Schattierungen werden u. U. beschnitten, wenn Sie mit Hilfe des PCL 6-Druckertreibers von HP den Drucker HP4050N verwenden. Sie sollten die entsprechenden Dateien mit dem PCL 5e-Treiber von HP drucken. [418321]

Umfließen
 Bei bestimmten Kombinationen von Vergrößerungsfaktor und Fenstergröße können Dokumente nicht umflossen werden. Acrobat Reader vergrößert das Dokument anhand eines beliebigen, anzeigbaren Zoomfaktors. [401685]

Kopie speichern
Acrobat Reader 5.0 enthält eine Menüoption und eine Werkzeugleistenschaltfläche namens "Kopie speichern". Wenn Sie eine Adobe PDF-Datei in Acrobat Reader 5.0 ändern, werden Ihre Änderungen nicht beibehalten, wenn Sie diesen Befehl verwenden. Es wird nur eine Kopie der ursprünglichen Adobe PDF-Datei gespeichert; das Speichern von Updates gehört nicht zu den Funktionen von Acrobat Reader.[414048]

SVG-Browser-Zusatzmodul (Scalable Vector Graphics) erfordert das Akzeptieren des Endbenutzerlizenzabkommens durch den Systemverwalter
Bei der Installation von Acrobat 5 auf einen Windows-Terminalserver oder ein gesperrtes NT/2000-System funktioniert das SVG-Browser-Zusatzmodul erst dann, wenn der Systemverwalter eine SVG-Site besucht, z. B. Adobes SVG-Demoseite unter:

http://www.adobe.de/svg/demos/

Bis dahin sieht der Benutzer ohne administrative Berechtigungen, der eine Web-Seite mit SVG besucht, jedoch verschiedene Dialogfelder zum Endbenutzerlizenzabkommen, und die Seite wird nicht korrekt angezeigt.[418523]

Web-Kauf unter Windows 2000
Unter Windows 2000 ist Service Pack 1 für Web-Kauf erforderlich, um ein Zip-Laufwerk als Gerät anzuzeigen:
"Bearbeiten>Grundeinstellungen>Allgemein>Web-Kauf>Weitere Kennungen auswählen". [390998]

Web-Kauf unter Windows 2000 in einem System mit mehreren Prozessoren
Wenn Sie ein elektronisches Buch mit Web-Kauf von einem Prozessor kaufen und später von einem anderen Prozessor aus darauf zugreifen wollen, wird Ihnen der Zugriff verweigert. Dies kann z. B. der Fall sein, wenn Sie das elektronische Buch bei einer Anmeldung erstehen und später versuchen, von einer anderen Anmeldung aus darauf zuzugreifen. Um dies zu vermeiden, sollten Sie beim Zugreifen auf das Buch die gleiche Anmeldung verwenden wie beim Kauf.[413006]

Web-Kauf benötigt die neusten Iomega-Treiber
Installieren Sie die neusten Iomega-Treiber für alle Jaz-, Clik!- oder Zip-Laufwerke, wenn Sie die Medien für diese Laufwerke als Kennungen für Web-Kauf verwenden wollen. Selbst wenn Ihr System die Laufwerke erkennt, erkennt Web-Kauf nur die Medien zur Lizenzierung, wenn die Iomega-Treiber installiert wurden.[413448]

Verwenden von Windows Internet Explorer
Wenn Sie Acrobat 4.x nach Acrobat Reader 5.x installieren, wird mit dem Browser das Acrobat 4.x ActiveX-Steuerelement verwendet. Aufgrund der Acrobat Reader 5.x-Registrierungseinstellung wird mit dem Internet Explorer jedesmal automatisch Acrobat 4 gestartet. Um dieses Problem zu vermeiden, sollten Sie Acrobat Reader 5.x vor der Installation von Acrobat 4.x deinstallieren. [366101]
