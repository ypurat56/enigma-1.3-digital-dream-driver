Ayuda sobre cómo utilizar el Visor SVG de Adobe®para Windows

Navegación

Panorámica: Mantenga pulsada la tecla Alt, haga clic y arrastre para crear una panorámica de la imagen SVG.

Zoom: Mantenga pulsada la tecla Ctrl y haga clic para acercar (reducir) la imagen que indica el puntero. Mantenga pulsada la tecla Ctrl, haga clic y arrastre para seleccionar el área que desea acercar. Mantenga pulsada la tecla Mayúscula para alejar (aumentar) la imagen. También puede utilizar los comandos de zoom del menú contextual.

Uso del menú contextual emergente

Con el botón derecho del ratón, haga clic en la imagen SVG para que aparezca el menú contextual. En este menú encontrará comandos y opciones para la imagen SVG.

Enlaces: Si abre el menú contextual en un enlace, podrá activarlo con las opciones Abrir o Abrir en nueva ventana.

Calidad: Al desactivar la opción Alta calidad, la imagen SVG aparecerá más rápido; sin embargo, la calidad será menor.

Animaciones: Use el comando Pausa para detener la animación de la imagen SVG. Este comando cambia a Reproducir cuando la imagen está en pausa. Use el comando Mudo para desactivar sonidos incluidos en la imagen SVG.

Búsqueda: Haga clic en Buscar para que aparezca el cuadro de diálogo de búsqueda. Escriba el texto que desea buscar, seleccionando las opciones para coincidencia de mayúsculas y minúsculas, palabras completas y el sentido en que realizará la búsqueda (hacia atrás, hacia adelante, etc.). Este comando realiza una búsqueda de textos en movimiento, oscurecidos o transparentes en todo el documento, independientemente del porcentaje de la imagen SVG que se encuentre visible. El resultado aparece destacado en una imagen panorámica (no aumentada) en pantalla.

Copia: Los comandos Copiar SVG y Guardar SVG como permiten copiar la imagen SVG en el portapapeles y guardarla en un archivo local. El comando Copiar coloca la imagen SVG tal y como aparece en el portapapeles en formato Unicode y rastrillado. A diferencia de Guardar SVG como, este comando realiza una copia del archivo SVG original. Al seleccionar el texto, el comando Copiar cambia a Copiar texto seleccionado.

Para más información sobre el Visor SVG de Adobe, consulte la sección sobre SVG en Adobe.com.
