Aide sur l'utilisation du module externe Afficheur SVG Adobe® pour Windows

Navigation

Affichage panoramique : cliquez sur l'image SVG tout en maintenant la touche Alt enfoncée, puis faites glisser l'image pour l'afficher en mode panoramique.

Zoom : cliquez à l'emplacement du point d'insertion du pointeur tout en maintenant la touche Contrôle enfoncée pour effectuer un zoom. Maintenez la touche Contrôle enfoncée, puis cliquez et faites glisser pour sélectionner la portion de la page sur laquelle effectuer un zoom avant. Pour réaliser un zoom arrière, maintenez également la touche Maj enfoncée. Vous pouvez également utiliser les commandes de zoom du menu déroulant.

Utilisation du menu contextuel

Cliquez avec le bouton droit de la souris sur la zone de l'image SVG pour ouvrir un menu contextuel qui regroupe les différentes options et commandes grâce auxquelles vous pouvez interagir avec l'image.

Liens : si vous faites apparaître le menu déroulant à partir d'un lien, choisissez la commande Ouvrir ou Ouvrir dans une nouvelle fenêtre pour activer le lien.

Qualité : vous pouvez désactiver l'option Qualité supérieure pour que l'image SVG s'affiche plus rapidement, mais avec une qualité inférieure.

Animations : utilisez la commande Pause pour interrompre une image SVG animée. Celle-ci devient la commande Lecture lorsque l'animation est interrompue. Vous pouvez désactiver les sons incorporés à l'image SVG grâce à la commande Sourdine.

Recherche : cliquez sur Rechercher pour ouvrir la boîte de dialogue de recherche. Vous pouvez alors saisir le texte à rechercher, sélectionner le sens de la recherche et préciser si vous souhaitez respecter la casse et faire porter la recherche sur tout ou partie du mot. Cette commande fonctionne également avec du texte transparent, masqué ou animé. La recherche s'applique à l'intégralité du document, quelle que soit la partie visible du fichier SVG. Le résultat de la recherche est mis en surbrillance, et l'affichage est recentré sur le résultat recherché (le résultat n'est touefois pas agrandi).

Copie : les commandes Copier le SVG et Enregistrer le SVG sous permettent de copier l'image SVG dans le presse-papiers et de l'enregistrer dans un fichier local. La commande Copier place le fichier SVG tel qu'il apparaît dans le presse-papiers, aux formats Unicode et pixellisé. Notez que cette commande est différente de la commande Enregistrer le SVG sous, qui consiste à effectuer une copie du fichier SVG d'origine. Lorsque vous sélectionnez du texte, la commande Copier se transforme automatiquement en commande Copier le texte sélectionné.

Pour de plus amples informations sur l'Afficheur SVG Adobe, consultez la rubrique consacrée au sur le site at Adobe.com.
